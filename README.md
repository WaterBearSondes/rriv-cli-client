# rriv-cli-client
CLI client for RRIV

# building
## Pop Linux
1. `sudo apt install cmake libserial-dev`
2. Enter the command `./compile.sh`

## Ubuntu
1. `sudo apt install cmake pkg-config cmake-data libserial-dev`
2. Enter the command `./compile.sh`

## Macos (currently unsupported)
1. ~Install homebrew if you have not already.~

# running
./build/rriv-cli
